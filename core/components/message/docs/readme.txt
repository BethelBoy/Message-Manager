Message manager is a CMP (custom manager page) and snippet for MODX Revolution.  The manager allows you to easily create messages that include audio, video, and pdf related documents. You can create groups that contain multiple messages inside them, keeping all your messages nicely organized.

Features:
	 - Create single messages to display on your site.
     - Upload audio, video, and pdf files for each message.
     - Create groups for each message to go in.
     - Easily display a certain group with the built in snippet call.

Install:
- Install via the MODX Revolution packagemanagment

Usage:
basic options creating a slide show

</div>
